PORTFOLIO

Objectifs du portfolio
Ce portfolio a été réalisé dans le cadre d’un projet académique.
Il a pour objectif de présenter mon profil, mon parcours académique,
mes compétences ainsi que mes expériences et projets.
Il constitue également une première vitrine professionnelle.

--------------------------------------------------

Contenu des pages

1. Page Accueil
Présentation générale du portfolio avec une introduction de mon profil
en tant qu’étudiante en L2 MIASHS.

2. Page À propos de moi
Présentation de mon parcours académique, de mes centres d’intérêt
et de mon orientation professionnelle.
Cette page inclut également :
- Mes compétences (programmation et développement web)
- Mes certificats
- Ma formation académique

3. Page Expériences
Présentation de mes expériences professionnelles :
- Stage chez PwC Turkey (Analyse de données et Intelligence artificielle)
- Stage à Borsa İstanbul
- Stage chez Rule Fragrances
Chaque expérience est accompagnée d’une description.

4. Page Tendances technologiques
Analyse d’une tendance technologique : le Data Analysis.
Cette section présente :
- La définition de la tendance
- Ses enjeux et son importance
- Son lien avec mon parcours académique et professionnel
- Un bilan et des perspectives

5. Page Projets
Présentation de plusieurs projets réalisés :
- Développement collaboratif d’un jeu de Blackjack en Java
- Conception et développement d’un site web personnel (portfolio)
- Participation à un projet de recherche à l’Université Koç
  (programme d’été KUSRP)

6. Page Contact
Page permettant de me contacter via :
- Une adresse e-mail
- Un numéro de téléphone
- Un formulaire de contact avec validation des champs
- Un lien vers mon profil LinkedIn
- Le téléchargement de mon CV

--------------------------------------------------

Instructions pour tester le portfolio localement

1. Dézippez l’archive du projet.
2. Ouvrez le fichier `index.html` dans un navigateur web compatible
   (Google Chrome, Mozilla Firefox, Microsoft Edge, etc.).
3. Naviguez entre les différentes sections à l’aide du menu.
4. Testez les fonctionnalités interactives :
   - Navigation fluide entre les sections
   - Changement d’onglets dans la section "À propos de moi"
   - Validation du formulaire de contact

--------------------------------------------------

Technologies utilisées

- HTML5 : Structure et contenu des pages.
- CSS3 : Mise en page, design et styles visuels.
- JavaScript : Interactivité (onglets dynamiques, validation du formulaire).
